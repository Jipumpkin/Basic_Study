선언 int num;

초기화 num = 5;

재할당 num = 6;

기본데이터 타입 8가지

1.논리형
    boolean : ture/false (1byte)
2.문자형
    char (2byte) 문자 한 개 , 유니코드
    저장할 떄 작은 따옴표'' 를 사용해서 저장해줘야 함
3.정수형
    1) byte (1byte)
    2) short (2byte)
    3) int (4byte)
    4) long (8byte)
4.실수형
    1) float (4byte)
    2) double (8byte)



** 형변환 (casting/cast)
1) 강제 형변환 (명시적 형변환)
큰 자료형을 작은 자료형으로 변환할때 사용.

2) 자동 형변환 (묵시적 형변환)
작은 자료형을 큰 자료형으로 변환할 때 사용한다.





연산자

피연산자(항)

피연산자와 연산자가 합쳐져 있는 것 (조건식)
> 식

기준 : 항의 개수
- 단항 연산자
    1) 증감 연산자(++,--)
        앞/뒤 num++,++num
        앞 : 연산이 먼저 이루어 진 다음 코드가 실행된다.
        뒤 : 실행코드가 동작한 다음 연산이 이루어진다. (행이 바뀌고 연산이 시행된다)

- 이항 연산자
    1) 산술 연산자 : +,-,*,/,%
        +는 문자열을 이어붙인다
        문자열+숫자는 숫자가 문자열로 형변환됨
    2) 비교 연산자 : > >= <= == !=
        결과값으로 boolean을 가진다
    3) 논리 연산자 : ! && ||
        피연산자(항)이 boolean type만 올 수 있다
        부정연산자 ! (false면 true)
        AND && (양쪽 모두 true면 true)
        OR || (하나 이상 true면 true)

- 삼항 연산자 (조건 연산자)
 ?:
조건식?실행문1:실행문2

조건식이 true면 실행문1를, false면 실행문2를 반환

조건식 : 결과값으로 t/f를 돌려주는 연산자와 피연산자가
합쳐진 구조
