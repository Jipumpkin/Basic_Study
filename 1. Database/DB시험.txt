데이터베이스란
- 여러 사용자가 공유하여 사용할 수 있도록 만들어진 통합해서 저장하는 운영 데이터의 집합이다.

DBMS(DataBase Managermenets System)
- 데이터베이스에 데이터 조작과 관리를 효율적으로 하기 위한
시스템 소프트웨어이다.
- Oracle, MySQL, MariaDB 등 많은 종류의 DBMS가 있다.

SQL이란?
Structure Query Langeage 의 약자로 데이터베이스에 접근하고 조작하기 위한 표준 언어이다.
- 데이터베이스에 데이터를 추출하고 조작하는데 사용하는 데이터 처리 언어이다.

SQL의 작성 순서
SELECT > FROM > WHERE > GROUP BY > HAVING > ORDER BY

SQL의 실행 순서
FROM > WHERE > GROUP BY > HAVING > SELECT > ORDER BY


SQL문법의 종류
DDL (Data Difinition Language) : 데이터 정의어
-- 세인업데

DCL (Data Control Language) : 데이터 제어어
-- GRANT, REVOKE, ROLE


TCL (Transaction Control Language) : 트랜잭션 제어어
-- ROLLBACK, COMMIT, SAVEPOINT
